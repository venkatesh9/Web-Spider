Project Name : Web Spider 

Instructions to run the program:

1) run gui.ss 
2) maximize the pop-up browser screen.
3) search required string.
4) left text box is search results of searched string.
5) right text box is prompt of searched string.
6) click on text box in search window to open required page in a browser.
7) click on text in prompt window to insert required string in text-feild.
8) use crawl web button  to search in web.
9) use clear button  to clear string in text field.
10) use next button to see results in next page.
11) use previous button  to see results in previous page.
